### Quick Deploy
```bash
# Clone and deploy
git clone https://github.com/adelfree2023-dev/KIMI

docker compose up -d

# Provision your first store
bun run cli provision \
  --subdomain=my-store \
  --plan=pro \
```

```bash

    "plan": "pro",
    "adminEmail": "admin@example.com",
    "storeName": "My Store"
  }'
```
